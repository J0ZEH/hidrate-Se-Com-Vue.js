Utilizando Vue.js, a aplicação proposta tem como objetivo desenvolver uma plataforma web para lembrar os usuários de beber água regularmente ao longo do dia e rastrear a quantidade de água ingerida. A aplicação é amigável, permitindo que os usuários estabeleçam metas personalizadas de ingestão de água e recebam lembretes ocasionais para garantir que permaneçam hidratados ao longo do dia. Através de uma interface intuitiva e interativa, os usuários podem informar a quantidade de água ingerida em cada lembrete recebido, o que permitirá o acompanhamento do seu consumo diário e o monitoramento do progresso em relação às metas estabelecidas.
